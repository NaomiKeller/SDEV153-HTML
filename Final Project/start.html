Arks-Layer Download

First, I will explain how to download PSO2 without use of the Japanese website. There is a third-party project called Arks Layer that has translated the game and streamlined the installation into the third-party launcher called Tweaker. Go to arks-layer.com. You can access Arks Layer anywhere on my site in the resources section of the left sidebar. On the home page, once you scroll down you should see a "Setup File" download button. Download and install PSO2 Tweaker.

Once it is finished installing, open Tweaker and it will automatically begin downloading the game, and then it will install the English patch. Follow the prompts and instructions, and ensure to create a directory for the game to install the game in once Tweaker asks for a directory. I recommend making a high level directory directly on a drive. Do not install the game somewhere like My Documents.


SEGA JP Registration Guide

Alright, that was the easy part. Now for the hard part: navigating a page and filling in a form that is in an entirely foreign language. It is hard to walk through this process without visuals, so here below I have a video that walks through the process. OR alternatively Arks Layer also has a guide on their website with a translated version of the form. You can view that here. It is really not as complicated as it seems, so I would recommend you look through text guide on Arks-Layer first. Just follow the instructions and use the translated screenshots they have there for reference as to what information to put in the form. The only tricky thing is the Japanese captcha. What I do is go to Google Translate, and use the handwrite function to emulate the characters as best as I can, and it usually works.
